devito.ir.equations.equation module¶

class devito.ir.equations.equation.LoweredEq(expr, subs=None)[source]¶

Bases: sympy.core.relational.Equality, devito.ir.equations.equation.EqMixin

A SymPy equation with associated iteration and data spaces.

All Function objects within expr get indexified and thus turned into objects of type types.Indexed.

The data space is an object of type DataSpace. It represents the data points accessed by the equation along each Dimension. The :class:`Dimension`s are extracted directly from the equation.

The iteration space is an object of type IterationSpace. It represents the iteration points along each Dimension that the equation may traverse with the guarantee that no out-of-bounds accesses will be performed.
